# MyLog_backend

> NodeJs ( Express + MongoDB )


### 상세내용

## 직장생활을 하면서 나만의 목표 ( Todo ) 성취하여 성장하기 위한 서비스  
  *   참고(Front) https://github.com/nullstudy/leaveday_web
## 기능

- 로그인 ( 소셜 - 카카오,구글 ) 
- 최종 Todo 날짜지정 (로그인 시)  
- main
  * 프로필 및 Mylog 프로그래스정보  
  * Calendar (메인)
    *  다이어리 보기 
    *  다이어리 등록
    *  다이어리 수정
- TodoList  
  * TodoList 생성
- MyDiary 
  *  게시판리스트
  *  다이어리 등록
  *  다이어리 수정
  *  다이어리 상세보기
